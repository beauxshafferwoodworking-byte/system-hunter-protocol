# SYSTEM // Hunter Protocol

A Solo-Leveling-inspired offline Progressive Web App workout tracker.

## What it does
- Daily quest generation
- Auto-adjusted workout type based on fatigue, sleep, lower back pain, and neck pain
- XP, levels, ranks, titles, streaks, stats
- Penalty quest system
- Offline PWA support
- iPhone home screen ready

## How to run locally
Open `index.html` in a browser.

## Best phone install method
For true PWA install behavior, host the folder somewhere simple:
- GitHub Pages
- Netlify
- Vercel
- Local network web server

Then open it on your iPhone in Safari:
Share button > Add to Home Screen.

## Note on iPhone notifications
This build is offline-first and does not include server-backed push notifications yet. iPhone PWA push notifications require Home Screen installation and additional web push support.
